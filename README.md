# Adotei

O **Adotei** é uma plataforma desenvolvida para centralizar e facilitar o processo de adoção e doação de animais de estimação. O projeto visa resolver a fragmentação de anúncios em redes sociais, oferecendo um ambiente seguro e organizado para conectar doadores e adotantes.

---

## Tecnologias
* **Banco de Dados:** MySQL
* **Backend:** Node.js
* **Frontend:** React Native
* **Prototipagem:** Figma
